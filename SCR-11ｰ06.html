問題

次の式を因数分解しなさい。

難易度 72PT
25a2 − 36b2
edit_note

あなたの解答

cancel 不正解
( 5a + 6b ) ( 5a − 6b )
解答 (5a + 6b)(5a − 6b)

ステップ1：平方の形に変形

$25a^2 = (5a)^2$ 、 $36b^2 = (6b)^2$ であることを確認します。

= (5a)² − (6b)²

ステップ2：公式の適用

公式 x² − y² = (x + y)(x − y) を利用して因数分解を行います。

= (5a + 6b)(5a − 6b)
